简单来说，VPN翻墙软件，会在你访问例如Youtube，P站等等网站的时候，先对你的数据进行加密，这样在经过中国网络的防火墙的时候，墙妈想要解析你包里的数据，但由于加密了解析不了，那就不会对你这个数据包进行拦截，这样，你就能安全放心的访问到国外的网站了。
